## 🏗️ Project Architecture

Resume Upload → Text Extraction → Resume Parsing → TF-IDF Vectorization → Cosine Similarity → Skill Gap Analysis → Candidate Ranking → Report Generation

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- NLTK
- Pandas & NumPy
- PyPDF2
- docx2txt
- Plotly
- FPDF
- Git & GitHub

## ⚙️ Workflow

1. Upload Resume (PDF/DOCX)
2. Enter Job Description
3. Extract & Clean Text
4. TF-IDF Vectorization
5. Cosine Similarity Matching
6. Skill Gap Analysis
7. Candidate Ranking
8. Export Report (CSV/PDF)

## 📌 Project Description

An AI-powered Resume Screening & Job Matching System that analyzes resumes, calculates job match scores using TF-IDF and Cosine Similarity, identifies skill gaps, ranks candidates, and generates recruiter-ready reports.
